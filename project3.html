The SharePoint React SPFX Learning Management System is a product developed for Microsoft to resell to other vendors. Hosted on SharePoint Online, it leverages built-in Active Directory roles for user management. The solution offers the following key features:

  • Private and public courses that can be divided per organization division
  • Integration with Microsoft Teams for webinars
  • Generation of a course completion certificate for users who complete all course content and assessments
  • Ability for admin users to create courses and assessments
  • A social sharing button to share the course certificate on LinkedIn

My role in the project involved:

  • Global SCSS styling to ensure a consistent look and feel throughout the application
  • Implementing React routing to manage the navigation between different pages
  • Developing all the administrative views and services to support the core functionality of the solution.
