I have developed a comprehensive Supplier Management Portal for Motus MMS that integrates seamlessly with Microsoft Dynamics 365. This solution has transformed their traditional supplier onboarding and management process into a fully digital experience, streamlining and automating key activities.

Some of the standout features of the solution include:

  • Token authentication through B2C
  • Utilization of Microsoft Forms Recognizer to scan supplier documents and automatically populate the system with relevant information.
  • A centralized administrative view through integration with Dynamics 365 CE.
  • Background checks on potential suppliers through the LexisNexis API.
  • Approval workflows to facilitate the supplier onboarding process.
  • IThe ability to sign SLA agreements electronically using Concord API
  • Ongoing supplier management capabilities

My key contributions to the project include:

  • Consulting client directly.
  • Solo presented the final solution to the executives of all the deviations at Motus MMS.
  • Training for all business partners how to use the entire solution.
  • Backend development of critical CRUD APIs and services.
  • Implementing Azure Key Vault with Managed Identity for secure configuration.
  • Implementing Azure Logic Apps for workflow management and notifications.
  • Building Azure Function Apps to handle all scheduled tasks.
  • Code optimization and refactoring.
  • Managing the DevOps and project pipeline to ensure smooth delivery.
