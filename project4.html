Turn your house into a restaurant web API inspired by Airbnb

Some of the standout features of the solution include:

  • Clean Architecture with a domain driven design.
  • Use MediatR to handle all Services.
  • Use the CQRS controller pattern.
  • Extend .Net Factories to handle errors and exceptions.
  • Allow any one to be able to turn there hose into a Restaurant through this API.
